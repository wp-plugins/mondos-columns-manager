A simple plugin to remove columns from the WordPress list view.

== Description ==

Once installed this plugin will allow you to remove core columns from the list view of Posts, Pages and Users.
Sometimes you might want to remove certain core columns from your WordPress install just to remove unwanted clutter.

== Installation ==

To install the plugin:

1. Upload `mondos-column-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Then got to the settings pages from the Dashboard > Settings > Columns Manager
4. By checking the checkboxes you will be able to disable the columns from the default list view.
